## 🚀 Getting Started

### Create Your First Mobile App
1. Go to JcalAI dashboard
2. Click "Create with AI"
3. Choose "Mobile App"
4. Describe your app:
   ```
   "Build a restaurant ordering app with menu browsing,
   cart, checkout, and order tracking"
   ```
5. Select framework (React Native or Flutter)
6. Click "Generate"
7. Customize in visual builder
8. Deploy to App Store
